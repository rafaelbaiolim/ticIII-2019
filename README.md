Execução 
=============
1. ```$ cd librec/bin```

2. ```./librec rec -exec -conf ../../conf/$nomeConf.properties -libjars ../lib/log4j-1.2.17.jar```

obs.: `$nomeConf.properties` são os arquivos disponíveis em ./conf/
